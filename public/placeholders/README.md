Placeholders for exercise demos live here. Add or replace files with GIFs or SVGs matching the filenames referenced in `data/workouts.json`.

Guidance:

- Keep files small to avoid bloating the repo. SVGs are preferred for placeholders.
- Filenames in `data/workouts.json` should match the files in this folder (for example, `DB_Bulgarian_Split_Squat.svg`).
- Files in this folder are served from `/placeholders/<file>` at runtime.
